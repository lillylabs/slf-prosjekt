=== slf-bike-rides-post-type ===
Contributors: raae

== Description ==

Makes posts into bike rides by renaming posts menu items and post object.
Makes categories into locations by renaming post menu items.

To add advanced custom fields functionality import acf.json.
* Adds extra fields to posts.
* Adds a logo field to categories.

To add taxonomies create the taxonomies with Simple Taxonomies
* name/slug | Label
* vanskelighetsgrad | Vanskelighetsgrad
* underlag | Underlag
* type-tur | Type tur

== Design ==

The design works with the slf-project theme.
